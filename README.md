# DBDCodes

A simple mobile companion for Dead by Daylight players.

## Why I built it
I developed this app to aid DBD players by keeping useful info like active codes, shrine updates, and news in one place.

## Features
- Active code references in one place
- Shrine of Secrets at a glance
- News and updates feed
- Lightweight, fast UI

## Tech stack
- Expo + React Native
- TypeScript

## Getting started
1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
   npx expo start
   ```
